# 🚀 SkillTrack Dashboard

A modern Django-based Skill Tracking Dashboard designed to help students and developers monitor their skills, projects, internships, and learning progress through a beautiful cyberpunk-inspired interface.

## 📌 Features

### 🏠 Dashboard

* Centralized overview of learning journey
* Modern dark red & black cyberpunk UI
* Responsive design

### 🐍 Skills Management

* Track technical skills
* Visual progress bars
* Dynamic data from Django Admin

### 💻 Projects Showcase

* Display completed projects
* Project descriptions and technologies used
* Portfolio-ready presentation

### 🚀 Internship Tracking

* Organize internship applications
* Monitor progress and opportunities

### 📊 Analytics Dashboard

* Total Skills Count
* Total Projects Count
* Average Skill Progress
* Dynamic calculations from database

### ⚙️ Admin Management

* Built using Django Admin
* Easy content management
* Secure authentication
